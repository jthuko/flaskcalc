How to enter equations in the math solver

Basic Operations

  • Make sure you put a space between operator. Example 8*x + 6
  • Follow BODMAS (brackets, orders, division and multiplication, addition and subtraction, for operations. Example(6+5)*3 is different from 6+5*3
  • Use ** instead of ^. Example x**2+4 instead of x^2+4
  • Use sqrt to find square root of a number. Example sqrt(64)
  • Use expand to simplify expressions. Example: expand((x + 2)*(x - 3)) gives you x**2−x−6
  • Use solve to solve equations such as quadratic equations. Example: solve(x**2 + x + 1) gives you the following results: [-1/2 - sqrt(3)*I/2, -1/2 + sqrt(3)*I/2]
  • Use factor for factorization. Example: factor(x**3 - x**2 + x - 1)), results: (x - 1)*(x**2 + 1
